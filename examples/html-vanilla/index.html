AssistantAILike

A lightweight, customizable AI-like chat assistant widget

Demo

Click the chat button in the bottom right corner to try the assistant!

This example demonstrates:

  • Custom knowledge base with multiple topics
  • Keyword mapping for fuzzy matching
  • Light/Dark theme support
  • Quick reply buttons
  • Typing indicator animation

Quick Start

Include the CSS and JS files in your HTML:

<link rel="stylesheet" href="assistant.min.css">
<script src="assistant.min.js"></script>
